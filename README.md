# Samyak Jain's 3D Portfolio

Welcome to my personal 3D portfolio! This interactive, visually immersive website is built to showcase my backend, frontend, and AI engineering skills, alongside cutting-edge creative web development techniques. 

Checkout the live version at: **[Samyakjain808.github.io](https://Samyakjain808.github.io/)**

## 🚀 The Vision

The goal of this portfolio is to bridge the gap between complex intelligent systems and seamless, beautiful frontend designs. It features 3D modeling, smooth scroll animations, and a structured layout optimized for performance.

## 💻 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **3D Rendering:** [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) & [Three.js](https://threejs.org/)
- **Animations:** [GSAP](https://gsap.com/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Deployment:** GitHub Pages

## 🛠 Features

- **Immersive 3D Models:** Integrated `.glb` files configured optimally for web rendering.
- **Dynamic Projects Section:** A carousel displaying full-stack projects, AI agents, and custom algorithms.
- **Timeline & Experience Layout:** Custom 3D-oriented waypoints showcasing milestones like Google Code Jam rankings, Bennett University, and research publications.
- **Modern UI Elements:** Glassmorphism, smooth scrolling, and custom SVG iconography.
- **Fully Responsive:** Perfectly scaled for both mobile and desktop screens.

## ⚙️ Running Locally

To run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Samyakjain808/Samyakjain808.github.io.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Samyakjain808.github.io
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
5. **View the application:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
